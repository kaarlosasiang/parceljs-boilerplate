## Quick Start

Clone the Repository
```bash
  git clone https://github.com/kaarlosasiang/parceljs-boilerplate.git
```

Install Dependencies
```bash
  cd parceljs-boilerplate
  npm install
```

Start Developement Server
```bash
  npm start
```

Build for Production
```bash
  npm run build
```
